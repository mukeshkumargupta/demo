# 01 - LLM Basics

Learn how to interact with LLMs using Ollama (local, no API key needed).

## Files

- `basic_completion.py` - Basic chat completion, system prompts, and streaming
- `prompt_engineering.py` - Zero-shot, few-shot, and chain-of-thought prompting
- `multi_turn_chat.py` - Interactive multi-turn conversation

## Setup

```bash
pip install -r requirements.txt
ollama pull llama3.2:1b
python basic_completion.py
```
