# 02 - Embeddings & RAG

Learn how to build Retrieval-Augmented Generation pipelines using local models.

## Files

- `embeddings_basics.py` - Generate embeddings and compute similarity
- `simple_rag.py` - Complete RAG pipeline: index, retrieve, generate

## Setup

```bash
pip install -r requirements.txt
ollama pull llama3.2:1b
ollama pull nomic-embed-text
python embeddings_basics.py
```
