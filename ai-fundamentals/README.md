# AI Fundamentals - Learning Project

A hands-on project to learn core concepts for building AI-enabled applications — from LLM basics to agents, MCP, workflows, and production deployment.

## Project Structure

```
ai-fundamentals/
├── README.md                  # This file
├── FUNDAMENTALS.md            # Complete learning guide
├── 01-llm-basics/             # LLM prompting & API usage
├── 02-embeddings-rag/         # Embeddings, vector DBs, RAG
├── 03-chains-workflows/       # Chaining LLM calls, workflows
├── 04-tools-function-calling/ # Function calling & tool use
├── 05-agents/                 # Autonomous AI agents
├── 06-mcp/                    # Model Context Protocol
├── 07-memory-state/           # Conversation memory & state
├── 08-evaluation/             # Testing & evaluating AI apps
└── 09-deployment/             # Production deployment patterns
```

## Getting Started

1. Read `FUNDAMENTALS.md` for the full learning roadmap
2. Work through each numbered folder in order
3. Each folder will contain its own README with exercises

## Prerequisites

- Python 3.10+
- Basic understanding of APIs and JSON
- An OpenAI API key (or any LLM provider)
