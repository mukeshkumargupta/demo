# 03 - Chains & Workflows

Learn how to combine multiple LLM calls into structured workflows.

## Files

- `sequential_chain.py` - Chain LLM calls where output feeds into the next
- `parallel_chain.py` - Run multiple LLM calls simultaneously
- `router_chain.py` - Classify input and route to specialized handlers

## Setup

```bash
pip install -r requirements.txt
python sequential_chain.py
```
